## 论文证据速查

### 体型分类有效性
**结论：预测价值弱。** Sheldon 体型三分法描述身材有用，但不能用来指导训练策略。减重反应更适合用体成分指标（肌肉量、脂肪分布）和实际代谢反应来解释。（Silventoinen 2021, AJHB; Esteve-Ibáñez 2025, Nutrients）

### 上交叉综合征运动干预
**结论：证据等级高。** 荟萃分析（22项研究）确认最有效方案 = 力量训练 + 拉伸 + 肩部专项练习。最有效具体动作：下巴后缩（chin tuck）、靠墙天使（wall angels）、弹力带划船、胸肌拉伸。（Sepehri 2024, BMC Musculoskeletal Disorders）

### 身体重组（热量缺口下同时减脂增肌）
**结论：对新手/超重者完全可行。** 关键条件：
- 蛋白 ≥ 2.2 g/kg LBM/天（≈ 1.6-2.4 g/kg 体重/天）
- 必须配合渐进式抗阻训练
- 基线体脂越高、训练经验越少 → 越容易同时增肌减脂
- 热量缺口 300-500 kcal/天
(Longland 2016, AJCN; ISSN Position Stand 2017; Vargas-Molina 2026, EJAP)

## 方案模板

### Mifflin-St Jeor 公式
```
男 BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 + 5
TDEE = BMR × 活动系数
活动系数：久坐1.2 / 轻度1.375 / 中度(3-5天/周)1.55 / 高度1.725
减脂热量 = TDEE - 400
```

### 宏量素分配（减脂期）
- 蛋白质：2.0-2.4 g/kg 体重（保肌肉，论文支持上限更高）
- 脂肪：0.8-1.0 g/kg（激素健康）
- 碳水：剩余热量填充

### 训练计划（每周4-5次，上下肢分化）

**上肢推日（周一/周四）**
- 杠铃卧推 4×8-10
- 哑铃过头推举 4×8-10
- 侧平举 4×12-15（肩宽核心！减脂期唯一怼容量的动作）
- 绳索下压 3×12
- 结尾：坡度快走 15min（坡度10-12，速度5-6km/h）

**下肢日（周二/周五）**
- 杠铃深蹲 4×8-10
- 罗马尼亚硬拉 4×8-10
- 腿举 3×10-12
- 小腿提踵 4×15
- 结尾：坡度快走 15min

**上肢拉日 + 体态矫正（周三）**
- 引体向上/高位下拉 4×8-10
- 坐姿划船 4×8-10
- 面拉 4×15（圆肩矫正核心，轻重量多次数！）
- 哑铃弯举 3×12
- 靠墙收下巴 5min
- 门框胸肌拉伸 3×30秒

**周六（可选）**：核心+弱项补充
**周日**：完全休息

### 体态矫正（每天，0成本）
- 收下巴靠墙站：每天5min，后脑勺/肩胛骨/屁股/脚跟四点贴墙
- 门框胸肌拉伸：每天3组×30秒
- 面拉：整合进训练日，轻重量高次数（15-20 reps）

### 内胚型饮食策略（关键！）
- 碳水定时：训练前1.5h吃全天碳水的50%，训练后30min快速补碳
- 其余两餐低碳水，靠蔬菜+蛋白质填饱
- 晚上8点后断碳，只吃蛋白质+蔬菜
- 禁忌：含糖饮料、啤酒、外卖炒饭/炒面

## 注意事项

1. **不要用体型三分法指导训练**：学术证据弱。用实际体成分数据驱动。
2. **蛋白质是身体重组的最关键可控变量**：宁高勿低。
3. **不要额外大量有氧**：抗阻训练本身已是有效减脂手段，过度有氧掉肌肉+激食欲。
4. **睡眠对内胚型减脂至关重要**：睡眠不足→皮质醇高→腹部脂肪顽固。
5. **新手福利期**：前3个月是同时减脂增肌的黄金窗口。
