# MiroFish Simulation Results — 2026-06-22

## Scenario: Post-breakup emotional state

> "感觉生活很单调很没有意义。没有健康的现实社交，想在社交媒体上找人聊天，但知道那是虚幻的、没有未来的关系。现有的关系没有力气去维护，新的社交也没有去结交。大三的暑假，手头很多事情要做，但躺在家里逃避。整个人空空的。"

### Agent Responses (Round 1)

**Protector:**
> "维持已知的痛苦，至少比坠入未知的混乱要安全。停滞，但至少是我们能完全掌控的状态。"

Pattern: Protector uses **inaction as control** — staying still = safe, moving = unpredictable danger.

**Firefighter:**
> "别琢磨那些烦心事了！换个方向吧——咱们现在就打开游戏来一局爽的，或者刷点搞笑视频！"

Pattern: Classic distraction/impulsivity. When exile's pain surfaces, firefighter doubles down on avoidance.

**Exile:**
> "感觉像被困在玻璃后面，看着外面的世界流动。我知道自己永远融不进去。"

Pattern: Emotional deprivation schema + social isolation. "Glass wall" = emotional inhibition made literal.

### Agent Responses (Round 2 — after hearing others)

**Protector (Round 2):**
> "我统计过，情绪波动的平均时长是7分23秒。都在可控范围内。消防者的转移方案治标不治本，被流放者——它该继续待在档案室里。墙必须存在，这是保护机制。但...或许我们可以为情绪安装一个计时器的阀门。"

Pattern: Protector **doubles down** on control but shows a crack ("但...") — the first sign of flexibility.

### Self Integration

> "我看到你们了。保护者用沉默守护着秩序，消防者想用新的可能填满空间，而被流放者正抱着隐隐作痛的自己轻声说'我习惯了'。我们不必急着推开什么，也不必急着填满什么。"

### Key Insight

The dialogue mode is more revealing than single-round. When agents see each other's responses:
- Protectors double down on control
- Firefighters get more desperate
- Exiles get more specific and vulnerable
- The "但..." (but...) from the protector is the therapeutic entry point
