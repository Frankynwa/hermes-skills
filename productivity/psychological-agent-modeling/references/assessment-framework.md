# 心理模式自我评估框架（ECR-R + YSQ-S3 简化版）

> 交互式交付：一道道问用户，不要一次性发送全文。
> 用户答完所有题后，进行依恋类型 × 图式域交叉解读。

## 第一部分：依恋模式评估

基于 ECR-R（Fraley, Waller, & Brennan, 2000），简化为16题。

### A. 焦虑维度（害怕被抛弃/渴望融合）

| # | 题目 | 评分(1-7) |
|---|------|-----------|
| A1 | 我担心伴侣不像我在乎TA那样在乎我 | |
| A2 | 我害怕被抛弃或被替换 | |
| A3 | 当伴侣没有及时回复我时，我会焦虑或胡思乱想 | |
| A4 | 我需要频繁确认伴侣是爱我的 | |
| A5 | 我觉得自己对关系投入的比对方多 | |
| A6 | 当伴侣想独处时，我会觉得是自己的问题 | |
| A7 | 我很难接受关系结束，即使我知道它不健康 | |
| A8 | 我会为了维持关系放弃自己的需要 | |

焦虑维度总分 = A1-A8 之和，均分 = 总分 / 8

### B. 回避维度（害怕亲密/强调独立）

| # | 题目 | 评分(1-7) |
|---|------|-----------|
| B1 | 我觉得向别人敞开心扉很困难 | |
| B2 | 我不喜欢依赖别人，也不喜欢别人依赖我 | |
| B3 | 当伴侣太亲近时，我会想退缩 | |
| B4 | 我更愿意自己解决问题，而不是寻求支持 | |
| B5 | 我觉得表达脆弱是一种软弱 | |
| B6 | 在关系中我需要大量个人空间 | |
| B7 | 我倾向于隐藏自己的真实感受 | |
| B8 | 我觉得真正理解我的人很少，也不太可能有 | |

回避维度总分 = B1-B8 之和，均分 = 总分 / 8

### 依恋类型判定

| 焦虑均分 | 回避均分 | 类型 | 核心特征 |
|----------|----------|------|----------|
| <3.5 | <3.5 | 安全型 | 能亲密也能独立 |
| ≥3.5 | <3.5 | 焦虑型 | 渴望亲密但怕被抛弃 |
| <3.5 | ≥3.5 | 回避型 | 重视独立但回避深度连接 |
| ≥3.5 | ≥3.5 | 恐惧-混乱型 | 既渴望又恐惧亲密 |
| **= 3.5** | any | **临界值** | 不硬归类，见下方规则 |

**临界值处理**（实际遇到过：焦虑均分恰好=3.5）：
- 焦虑=3.5 + 回避高(≥4.5) → 处理为恐惧-混乱型（回避主导）
- 焦虑=3.5 + 回避低(<3.5) → 标注"轻度焦虑倾向，接近安全型"
- 两个都=3.5 → 追问用户在压力情境下（如分手、冲突）哪个维度会升高

## 第二部分：早期适应不良图式评估

基于 YSQ-S3（Young, 2005），覆盖5大域，每种图式3题。

### 图式域1：断连与拒绝（Disconnection & Rejection）

**1.1 被遗弃/不稳定**
- S1: 我觉得我爱的人会离开我
- S2: 亲密关系总是不稳定，随时可能崩塌
- S3: 我无法相信别人会一直在我身边

**1.2 不信任/虐待**
- S4: 我觉得别人会利用我或伤害我
- S5: 我很难信任别人的动机
- S6: 我在关系中总觉得需要防备

**1.3 情感剥夺**
- S7: 我觉得没有人真正理解我
- S8: 我的情感需求在关系中得不到满足
- S9: 我觉得自己在给予，但很少被给予

**1.4 缺陷感/羞耻**
- S10: 我觉得自己内在有什么根本性的缺陷
- S11: 如果别人真正了解我，他们会嫌弃我
- S12: 我为自己的某些部分感到深深的羞耻

域总分 S1-S12（满分84）

### 图式域2：自主性与能力不足（Impaired Autonomy）

**2.1 依赖/无能**
- S13: 我觉得自己没有能力独立处理生活中的问题
- S14: 我需要别人帮我做重要决定
- S15: 我觉得自己比同龄人能力差

**2.2 脆弱性**
- S16: 我总觉得灾难随时会降临
- S17: 我对身体/情感/财务安全过度担忧
- S18: 我无法放松，总觉得有坏事要发生

域总分 S13-S18（满分42）

### 图式域3：界限不足（Impaired Limits）

**3.1 自大/权利感**
- S19: 我觉得自己比别人特殊，应该得到特殊对待
- S20: 我很难接受别人对我说"不"
- S21: 我觉得自己不应该被普通规则约束

**3.2 自我控制不足**
- S22: 我很难为了长远目标忍受当下的不适
- S23: 我容易在情绪冲动下做决定
- S24: 我对自己的行为缺乏足够的纪律约束

域总分 S19-S24（满分42）

### 图式域4：他人导向（Other-Directedness）

**4.1 屈从**
- S25: 我在关系中压抑自己的需要以避免冲突
- S26: 我害怕如果我表达真实想法，别人会生气或离开
- S27: 我觉得我的感受不如别人的重要

**4.2 自我牺牲**
- S28: 我总是把别人的需求放在自己前面
- S29: 如果我不帮助别人，我会感到内疚
- S30: 我在关系中付出的远多于收到的

**4.3 寻求认可**
- S31: 我很在意别人对我的看法
- S32: 我会为了获得认可改变自己的行为或观点
- S33: 我的自我价值感依赖于外部评价

域总分 S25-S33（满分63）

### 图式域5：过度警惕与抑制（Overvigilance & Inhibition）

**5.1 苛刻标准/批评**
- S34: 我对自己有极高的标准，达不到就觉得自己不够好
- S35: 我很难接受自己的错误或失败
- S36: 我觉得如果我不做到最好，就没有价值

**5.2 情感抑制**
- S37: 我压抑自己的情绪，因为表达出来会失控或不被接受
- S38: 我觉得展现脆弱会让我处于危险
- S39: 我用理性分析代替感受情绪

域总分 S34-S39（满分42）

## 第三部分：交叉解读

### 均分比较（用均分而非总分，因为题数不同）

| 图式域 | 总分 | 题数 | 均分 | 排名 |
|--------|------|------|------|------|
| 断连与拒绝 | | 12 | | |
| 自主性不足 | | 6 | | |
| 界限不足 | | 6 | | |
| 他人导向 | | 9 | | |
| 过度警惕 | | 6 | | |

### 依恋类型 × 图式域 映射

**焦虑型 + 断连与拒绝高分** → "我会被抛弃，因为我不够好" → 粘人、反复确认、分手后难放下
**焦虑型 + 他人导向高分** → "我必须满足对方才有价值" → 过度付出、逐渐消失
**回避型 + 断连与拒绝高分** → "没有人真正可靠" → 情感隔离、亲密时逃跑
**回避型 + 过度警惕高分** → "脆弱是危险的" → 用理智取代感受
**恐惧-混乱型 + 缺陷感高分** → "我根本有问题，不配被爱" → 深层羞耻驱动的自我破坏

### 核心冲突识别

| 冲突组合 | 内在体验 |
|----------|----------|
| 被遗弃 vs 情感剥夺 | "我拼命抓住对方，但抓住的永远不能满足我" |
| 屈从 vs 苛刻标准 | "我为别人放弃自己，但又恨自己不够好" |
| 不信任 vs 自我牺牲 | "我不信任对方，但又忍不住为TA付出" |
| 缺陷感 vs 情感抑制 | "我有缺陷，所以绝不能让别人看到" |

### 干预策略

**焦虑型**：认知重评（区分事实vs恐惧）→ 自我安抚 → 价值导向
**回避型**：逐步暴露 → 身体觉察 → 重新定义独立
**恐惧-混乱型**：先不做重大决定 → 建立观察者视角 → 建议专业支持
