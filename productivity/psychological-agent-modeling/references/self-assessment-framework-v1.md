# Assessment Instrument v1

Full document: ~/projects/psychology-assessment/心理模式自我评估框架.md

## Structure

### Part 1: Attachment (ECR-R based)
- Anxiety dimension: 8 items (A1-A8), 1-7 scale
- Avoidance dimension: 8 items (B1-B8), 1-7 scale
- Scoring: mean per dimension, threshold 3.5
- Types: Secure (L/L), Anxious (H/L), Avoidant (L/H), Fearful-Disorganized (H/H)

### Part 2: Early Maladaptive Schemas (YSQ-S3 based)
39 items across 5 domains, 3 items per schema, 1-7 scale:

Domain 1: Disconnection & Rejection (12 items, S1-S12)
  - Abandonment, Mistrust, Emotional Deprivation, Defectiveness

Domain 2: Impaired Autonomy (6 items, S13-S18)
  - Dependence/Incompetence, Vulnerability to Harm

Domain 3: Impaired Limits (6 items, S19-S24)
  - Entitlement, Insufficient Self-Control

Domain 4: Other-Directedness (9 items, S25-S33)
  - Subjugation, Self-Sacrifice, Approval-Seeking

Domain 5: Overvigilance & Inhibition (6 items, S34-S39)
  - Unrelenting Standards, Emotional Inhibition

### Part 3: Cross-Interpretation
- Attachment type × Top schema domains → personalized "inner script"
- Core conflict identification (schema clashes)
- Intervention strategy map per pattern

### Part 4: Action Plan
- Immediate (this week): complete assessment, identify patterns
- Short-term (1-4 weeks): schema trigger journaling, start one strategy
- Medium-term (1-3 months): professional help if needed, corrective emotional experiences

## Key Design Decisions
- Use mean scores (not raw sums) for cross-domain comparison
- Ask ONE question at a time in interactive delivery
- Give dimension-level results after each section
- Full interpretation only after all data collected
- Always cross-reference attachment with schemas (they interact)
